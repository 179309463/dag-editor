# DAG图形设计器
DAG editor based on GoJS, as general task-workflow(任务编排) OR micro-service-workflow(微服务编排) designer。简易DAG图形设计器，使用拖拽方式对应用自动化编排。包括Host、Docker应用等。此项目为私有云平台子项目DEMO。
